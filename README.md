## 게임 설명
- 게임 컨셉  
게임은 공포 장르의 컨셉이며 보석을 모아 탈출구를 열어서 탈출하면 성공하는 게임이다.
- 게임 조작법  
W, A, S, D를 사용하여 이동하고 마우스 조작으로 좌, 우로 시점을 돌릴 수 있다.
