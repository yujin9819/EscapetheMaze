## 게임 컨셉  
게임은 공포 장르의 컨셉이며 좀비를 피해 보석을 모으면 탈출구가 열린다. 탈출구로 탈출하는 게임이다.
## 게임 조작 
W, A, S, D를 사용하여 이동하고 마우스 조작으로 좌, 우 시점을 돌릴 수 있다.
